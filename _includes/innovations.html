oZone brings in useful innovations to ZoneMinder

Such As:

  • zmNinja, the rockstar multi-platform ZoneMinder app. No other home security app is as cool as this one.
  • Alexa integration (sneak preview)
  • ZoneMinder as a library for 3rd party innovation (work in progress)
  • Simplified UI that you can relate to (work in progress)
  • Real-time event server with web-sockets - build your own event processor! (already in production!)
  • Presence aware (work in progress, exists in part)
  • And hopefully many more to come!



How do these innovations help ZoneMinder?

  • More adoption and awareness of ZoneMinder. Making it simpler and easier can't hurt?
  • Almost everything oZone will work on will also be under GPL, especially those features that build on top of ZoneMinder. If these innocations make sense for ZoneMinder, they will be merged!
